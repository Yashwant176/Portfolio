## Installation

Clone the repository:

```bash
git clone https://github.com/Yashwant176/Portfolio.git
```

Navigate to the project folder:

```bash
cd my-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```
